# Meal Tracker Masterplan

## North Star

Meal Tracker helps Sara quickly remember which meals to reorder, which were just okay, and which to avoid.

The app should feel fast, clean, practical, and calm. The main job is not food discovery or nutrition tracking; it is reliable personal meal memory.

## Core Outcome

Sara can:

- Add meals to a personal meal library.
- Rate meals as Favourite, Fine, Avoid, or Not rated.
- Search and filter the meal list quickly.
- Check notes before ordering again.
- See which dinner meals were ordered by Thursday week.
- See meal order counts and full order history.
- Use the app comfortably on this PC in a minimal dark interface.

## MVP Scope

Included:

- Local personal app.
- SQLite persistence.
- Image URL, paste, drag/drop, and file upload support.
- Ingredients and nutritional information stored as pasted/uploaded reference images.
- Home, Meals, Orders, Add/Edit, and Detail screens.
- Removed status to hide meals from default views.

Excluded:

- Backup/export.
- Website scraping or automated import.
- AI parsing.
- Nutrition analysis or automatic extraction from nutrition labels.
- Meal recommendations.
- Calendar planning.
- Weekly breakfast/lunch planning.
- User accounts or sharing.
- Notifications.
- Advanced analytics.

## Design Direction

- Dark mode first.
- Compact, scannable, and efficient.
- Use Inter/system typography, dark surfaces, subtle borders, and emerald accents.
- Avoid decorative landing-page sections.
- Keep the meal list as the primary working screen.
